
# BLINKIT SALES ANALYSIS

This project analyzes Blinkit grocery sales data to uncover trends, patterns, and business insights.


## 🚀 PROJECT DESCRIPTION
This project focuses on analyzing Blinkit grocery sales data to uncover meaningful insights and business trends.  
The goal is to understand how factors like outlet type, location, and product categories impact overall sales performance.  

By using Excel and data analysis techniques, this project transforms raw data into actionable insights for better decision-making.
## 🔍 KEY INSIGHTS

### 📈 Sales Insights
- Tier 1 cities generate the highest revenue  
- Medium-sized outlets contribute maximum sales  
- Small outlets show comparatively lower performance  

### 🛍️ Product Insights
- Certain product categories dominate overall sales  
- High-demand products maintain consistent performance  

### 🏪 Outlet Insights
- Supermarket-type outlets outperform grocery stores  
- Outlet size directly affects revenue generation
